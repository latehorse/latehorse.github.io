梦想虽晚，“码”不停蹄

  • 晚码科技
  • 2018-05-16
  • 晚码

五月就这样算是结束了，三天小假期，算是休息一下，却还是躲不过内心的挣扎，再坐回电脑前敲敲打打，倒不是公司的任务有多重，只是我耐不住这如此滞后的学习进度，于是，搜索，编写，测试，总结，一个一个小小的应用，简单甚至粗糙，可我乐此不疲。

窗外沸沸嚷嚷，五月过后便又到六月了，又是一年毕业季，吵吵闹闹的年轻，神采飞扬的青春，这沸腾的热情和活力，我唏嘘不已，已经毕业三年了，可三年前的那份骄傲和意气风发我还清楚的记得：自始仰笑出门去，日行千里走江湖。

我终究还是没能日行千里，两年多的工作生涯，属于基层服务类，需要经常上门的那种，只是着实让我为这世间百态感慨万千，这形形色色的人，那千奇百怪的事，不懂，不说，懂了，不说。

在 it 行业，进入程序开发行列，有些窃喜，也有些胆怯，我终究只能算是个半路出家，虽然不能说是逼上梁山，但也是有些穷途末路的意味，我常常在奇怪，我一直向往着，却从未想过将它变为现实，可当走投无路的时候，却还是它给了我置之死地而后生的勇气，这究竟是在追寻，还是被恩赐，我三心二意，它却不离不弃，我就像个浪荡子，幸运的碰到温暖的归宿。

新入行的菜鸟，确然有太多不懂，不过也算有颗应用物理人的头脑，逻辑和计算不能成为大问题，而设计和理念，以及太多原理和概念却成了我的软肋。常有朋友跟我说，甚至培训班的老师也这么说过，这些不过是些工具，你会用就可以了，现代社会技术发展这么快，各种语言也是更新频繁，只要会用，能用它来做东西，完成公司的任务，公司发给你钱就可以了，事实上，我初始阶段对这种观点还是很信服的。可是现在，我却不再这样想了。

每一个行业自然都有它的精髓和魅力，虽然可能达不到一定的高度，但是真的希望可以在这条路上走的更远，所以我希望更系统的了解它，希望了解它的来源和历史，而不是单调的调用工具，更希望能懂的是它的理念和思维，我始终坚信一个道理，就是只有真正有底蕴、有内涵的人，才能走的更远，那积淀下来有着最醇的芬芳，就像一坛老酒，愈久弥香…

开始看这方面的文章，开始关注这方面的消息，这段时间来给我留下印象最深的有两个人，一个是之前提到过的我来博客的源动力，jennifer dewalt，那个设计女孩，一个是季逸超，90后少年，有头脑有激情的小伙子，实话说让我感触很深的不是它的成就，而是他可以为自己是程序员老鸟而骄傲的那种自然，在一个行业就这么坚持着，如果是因为生活的压力走上这条行业，并且越做越大，越坐越久我可以理解，但是一个少年，可以就这么坚持着，并让它成功的成为了自己的事业，那种疯狂的坚守才是让我动容的根本。

我羡慕他们的故事，佩服他们的成就，相信很多人会跟我一样为他们的成功喝彩，毕竟这都是些励志的典型，而我在默默呐喊的同时，还有种淡淡的甜蜜，小小的幸福感，因为我也选择了这条路，我会这条路上走很远，这是属于我们的故事，有他，有她，不知道有天会不会有我。

梦想很遥远，可我愿马不停蹄的前行，为了梦想而奋斗总是痛苦的，又总是快乐的，我知道，可我来了，这种感觉很幸福。

总结下五月来自己的学习所得：（实话说，这开始阶段，真的没得总结，就那么点东西，但将就着来吧，慢慢进步，平常心啊，呼…）
